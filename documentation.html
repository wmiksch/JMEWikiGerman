Diese Doku liefert eine Anleitung für die Installation und Konfiguration, jME Programmiertutorials und andere Information die ihnen helfen werden ihr Gaming-projekt zum laufen zu bringen. Sie können die Suchfunktion am oberen Ende jeder Wikiseite benutzen um das gesamte Wiki zu durchforsten.

Sie sind auch dazu angehalten Verbesserungsvorschläge bei dieser Übersetzung des "'Wikis'" mit dem Menü "'Edit'" und “Add” zu geben. Sie müssen in GitHub eingeloggt sein um das wiki zu bearbeiten. Hier können sie mehr über das Editieren und Beitragen erfahren.

Das ist ein Warnhinweis. Es wird an verschiedenen Stellen solche im Wiki geben. Unter Warnhinweise finden sie die Bedeutung dieser.

Forum

Das Forum ist die erste Anlaufstelle um Fehler zu melden, Beiträge zu leisten, den Wiki Inhalt zu diskutieren oder auch sonst, wenn sie die Antworten auf ihre Fragen im Wiki und den Forenarchiv nicht finden. Sie werden dort eine freundliche und einladende Atmosphäre vorfinden, aber bevor sie posten beachten sie die folgenden 2 Regeln.

  1. Für Fragen die Quelltext beinhalten, lernen sie wie man Programmblöcke einfügt. Sie werden sonst statt einer Anwort höchstens eine Rüge erhalten. In den Github Tipps steht wie sie Links in ihre Beiträge einfügen können.

  2. Hochladen von Bildern ist gesperrt mit der Ausnahme eines 50k großen Forum-Avatars.

* The SDK creates Ant-based projects that any Java IDE can import. We recommend users of other IDEs to also download the jMonkeyEngine SDK and choose File ▸ Import Project ▸ External Project Assets to create a codeless project for managing assets only. This way you can code in the IDE of your choice, and use the SDK to convert your models to .j3o format.
